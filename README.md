# Login & Information Web Project

## Overview
This is a simple **web project** built using **HTML, CSS, and JavaScript**.  
It consists of two main pages:  

1. **Login Page**: Allows users to enter a username and password (demo login, any input works).  
2. **Information Page**: Displays general web development information with images and a video for an interactive experience.

---

## Project Demo
You can try the project live here: [Login Project](https://abdullahalassal-maker.github.io/login-project/)

---

## Features
- **Responsive Login Page** with styled form and button interactions.  
- **Demo Login**: No real authentication required, just enter any username and password.  
- **Information Page** with:  
  - Images and videos embedded.  
  - Clean, responsive design with modern CSS.  

---

## Technologies Used
- **HTML5**  
- **CSS3** (flexbox, gradients, shadows, and responsive design)  
- **JavaScript** (localStorage for storing username and page navigation)
